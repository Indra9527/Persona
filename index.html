Parking Meter Interface Persona

Introduction

The parking meter is something I often see on the side of the road but never use. Once I was walking along and a tourist asked me how to use a meter, unfortunately, I haven't used one either.

Later I'm curious about how it works and tried by myself. I found the user interaction was not very sensitive, and the operation is not sequence so that take me some time. Therefore I started my exploration.




Interview for users

  1. Have you tried looking for free parking nearby?
    • Basically, try to find a free location.
    • Running out of time or too far so had to stop at the paid area.
  2. Do you prefer to use cards or coins?
    • Depends on personal habits, but cards may be more convenient nowadays
    • But the card has limitations in some machines (minimum spend, only accept credit cards, etc.)
    • Choose coins if capable and enough.
  3. When was the last time you used a parking meter?
    • Basically, not very frequent, occasionally used.
    • One elder guy is using it for the first time.
  4. If this is your first time using a parking meter, how would you like it to be more efficient?
    • hope the font on the electronic screen can be clearer.
    • list the process steps to be simpler, after all, in a hurry.
    • Buttons work well!
